# Add System Tray Support

## Why

当前应用关闭窗口即退出程序。为了提升用户体验，需要增加可选的"最小化到系统托盘"功能：
- 用户点击关闭按钮时，应用隐藏到托盘而非退出
- 托盘图标提供快速操作菜单（显示/退出）
- 该功能默认关闭，用户可自行启用

## What Changes

### 新增功能

1. **系统托盘集成**：添加托盘图标和右键菜单
2. **关闭行为配置**：可选择关闭时隐藏到托盘或直接退出
3. **设置持久化**：保存用户的托盘偏好设置

### 相关组件

- `internal/tray` - 托盘模块（新增）
- `internal/config` - 配置管理模块（新增）
- `app.go` - 添加托盘相关方法
- `main.go` - 初始化托盘
- `frontend` - 设置界面

## Impact

### 新增 Specs

- `system-tray` - 系统托盘功能规范

### 修改 Specs

无（specs 目录当前为空）

## Risks

- **Windows 特有**：托盘功能主要针对 Windows，其他平台行为可能不同
- **Wails v2 限制**：v2 的托盘 API 需要手动管理窗口可见性（v3 更完善）
